# 项目简介

本项目是在eladmin的基础上进行二次开发的项目，隶属NT1组内专用项目模板

演示地址：http://eladmin.remember5.top

## 规定

### 分支定义
单独的模块，请单独开启分支
* `master`分支做为发布分支
* `dev`分支做为开发合并分支
* `api`包含一个独立api的项目
